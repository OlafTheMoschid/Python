# Видеохостинг

## возможности

- просмотр и управление видео-потоком (остановка, пауза, воспроизведение, перемотка) без регистрации аккаунта
- загрузка видео на хост и его удаление оттуда
- поиск видео по названию в поисковой строке
- поддержка видео кодировоки mp4
- изменение и удаление видео
- изменение и удаление аккаунта 

## сборка проекта

Чтобы собрать проект, необходимо установить библиотеки в файле requirements.txt командой:
pip install -r requirements.txt

Чтобы запустить проект, необходимо запустить файл app.py в корневой папке проекта с помощью инерпретатора python 3.9.2

## управление

На главной странице можно зарегистрироваться или войти под своим аккаунтом, нажав на соответствующие кнопки
Если пользователь авторизован, он может загрузить видео с жесткого диска соответствующей кнопкой
Также на главной странице размещены загруженные видео в виде таблице, при нажатии на кнопку Watch откроется страница с выбранным видео.

На этой странице будет размещено видео.
Управление видео-потоком стандартно: остановка, пауза, воспроизведение, перемотка
Если текущий пользователь является автором видео, он сможет переименовать или удалить видео.

Чтобы найти видео по названию или его части, можно в перейти на страницу search/<часть_названия>, где будут все видео соответствующие поисковому запросу
