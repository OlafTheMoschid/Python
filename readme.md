# АудиоПлейер 

## возможности

- управление аудио-потоком (остановка, пауза, воспроизведение)
- добавление в таблицу музыки и ее удаление оттуда
- поддержка drag & drop при добавлении музыки в таблицу
- поддержка аудио кодировок: mp3, ogg, wav
- сохранение состояния после закрытия приложения 

## сборка проекта

Чтобы собрать проект, необходимо установить библиотеки в файле requirements.txt командой:
pip install -r requirements.txt

Чтобы запустить standalone-приложение, скачайте все архивы main.zip и main.z01 и распакуйте архив main.zip 

## управление

В контекстном меню file во вкладке open... открывается диалог с выбором аудиофайла.
Также приложение поддерживает технологию drag & drop: аудиофайл(ы) можно «перетащить» в таблицу.
Файл(ы) окажутся в конце списка.

Приложение позволяет настраивать громкость малым ползунком и перемещаться по аудиофайлу большим ползунком. 
Также большой ползунок показывает какой процент файла проигран.

Кнопки Stop, Play, Pause позволяют останавливать, запускать и ставить на паузу аудиофайл.
Кнопки Prev, Next позволяют перемещаться по файлам в таблице.

Нажатие кнопки Delete (на клавиатуре) удаляет выбранный аудиофайл из таблицы.
